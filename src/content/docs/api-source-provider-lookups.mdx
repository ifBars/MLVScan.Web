        Source provider lookups let an integration find public reports MLVScan has already indexed
        from supported providers such as Nexus Mods and Thunderstore. This is the same report-discovery
        path used by MLVScan.Bot lookup commands and source-linked public report pages.
      </p>

      <div className="rounded-lg border border-amber-900/50 bg-amber-950/20 p-4 mb-8">
        <h3 className="mt-0 text-amber-200 font-semibold mb-2">Lookup does not scan or crawl providers</h3>
        <p className="text-sm text-gray-300">
          Source lookup searches MLVScan&apos;s existing source index. It does not upload bytes, start a
          scan, or query the whole Nexus Mods or Thunderstore catalog. If a mod has never been scanned
          through a source-linked workflow, the lookup can return no results even if the mod exists on

      <h2 id="when-to-use-source-lookup">When to use source lookup</h2>
      <ul className="list-disc pl-6 space-y-2 text-gray-300 mb-8">
        <li>A mod manager wants to show the latest known MLVScan report for a provider mod page.</li>
        <li>A telemetry backend wants to enrich newly observed mods with an existing public report when one is already available.</li>
        <li>A moderation tool knows a Nexus mod ID or Thunderstore package name and wants recent source-linked report handles.</li>
      </ul>

      <h2 id="identity-model">Identity model</h2>
      <p className="text-gray-300 mb-4">
        SDK callers use one typed <code>MlvScanSource</code> value so provider and game cannot be mixed
        accidentally. The wrapper maps that source to the API&apos;s provider and game filters.

      <div className="overflow-x-auto mb-8">
        <table className="min-w-full text-sm text-gray-300">
          <thead className="text-left text-gray-100">

              <td className="border-b border-gray-800 py-2"><code>schedule1</code></td>

var client = new MlvScanClient(Environment.GetEnvironmentVariable("MLVSCAN_API_KEY"));

var sources = await client.ListSourcesAsync(new MlvScanSourceLookupOptions
{
    Source = MlvScanSource.NexusModsSchedule1,
    ModId = "1194",
    IncludeReports = true,
    Limit = 1,
});

var source = sources.FirstOrDefault();
if (source?.LatestSubmissionId is null)
{
    Console.WriteLine("No source-linked MLVScan report is indexed for that Nexus mod yet.");
    return;
}

var publicReport = await client.GetPublicReportAsync(source.LatestSubmissionId);

Console.WriteLine(publicReport.Classification);
Console.WriteLine(publicReport.Headline);
Console.WriteLine(publicReport.Summary);`}</CodeBlock>

    IncludeReports = true,
    Limit = 5,
});

foreach (var source in sources)

    Console.WriteLine(
        $"{source.DisplayName} | mod id: {source.ModId} | latest submission: {source.LatestSubmissionId}");

var bestMatch = sources.FirstOrDefault();
if (bestMatch?.LatestSubmissionId is not null)
{
    var publicReport = await client.GetPublicReportAsync(bestMatch.LatestSubmissionId);
    Console.WriteLine(publicReport.Headline);
}`}</CodeBlock>

})

  query: "S1API",
